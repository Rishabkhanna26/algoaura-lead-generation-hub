# AlgoAura Lead Generation Hub

This project is now a **Next.js App Router** app using **JavaScript/JSX** and **Tailwind CSS**.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build for production

```bash
npm run build
npm run start
```

## Structure

- `app/` - routes, layout, global styles, and API route
- `components/` - reusable UI/section components
- `lib/` - utility helpers
- `styles/` - theme tokens
- `public/` - static assets
